# 🚀 Focus Vocab

Một con app học từ vựng do chính tui build để tự học (thực ra là vide code lol).

Không quảng cáo.  
Không màu mè dopamine rẻ tiền.  
Chỉ có học – nhớ – quên – rồi kéo lại.

---

## 🧠 Nó làm được gì?

- 📚 Quản lý từ vựng bằng file JSON
- 🃏 Flashcard để ôn nhanh
- ⌨️ Terminal mode để gõ lại (Active Recall thật sự)
- 📊 Phân loại: Từ mới / Tạm nhớ / Nhớ tốt / Chưa nhớ
- 🔥 Memory Decay Engine (không ôn là tụt hạng)
- 🛡 Resilience: ôn lại đúng thì lần sau bị phạt nhẹ hơn
- 🏆 Rank system theo số từ đã thuộc

---

## 🎯 Triết lý

- Nhìn quen không có nghĩa là thuộc.
- Không gõ được = chưa nhớ.
- Không ôn = sẽ quên.

App này ép não hoạt động thật, không tự lừa mình.

---

## 🗂 Cách dùng

1. Import file JSON từ vựng
2. Chọn chế độ học:
   - Flashcard (ôn nhanh)
   - Terminal (gõ lại từ)
3. Học đều mỗi ngày
4. Đừng để Memory Engine đẩy lùi bạn 😏
